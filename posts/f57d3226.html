divmod()

divmod()是一个内置函数，它接受两个数字作为参数，并返回一个包含商和除法运算的余数的元组。

  • If x and y are integers, the return value is (x // y, x % y)
  • If x or y is a float, the result is (q, x % y), where q is the whole part of the quotient.
